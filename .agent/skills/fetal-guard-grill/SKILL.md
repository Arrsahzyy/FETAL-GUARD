# Fetal Guard Grill Skill

Sebelum menulis kode atau mengubah file, lakukan sesi klarifikasi.

## Required Behavior

1. Jelaskan ulang permintaan user dalam bahasa Indonesia sederhana.
2. Identifikasi subsystem terdampak:
   - embedded/ESP32
   - BLE/mobile app
   - WiFi/cloud
   - API/backend
   - dashboard/frontend
   - AI/ML pipeline
   - documentation/proposal
   - validation/testing
3. Pisahkan:
   - fakta yang sudah diketahui
   - asumsi
   - pertanyaan terbuka
   - risiko
4. Jangan membuat klaim medis tanpa bukti validasi.
5. Akhiri dengan rekomendasi langkah berikutnya.
