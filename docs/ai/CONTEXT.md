# Fetal Guard Context

## Deskripsi Singkat

Fetal Guard adalah prototype alat portabel untuk membantu skrining awal risiko pada ibu hamil. Fokus project meliputi pembacaan sensor, konektivitas BLE/WiFi/cloud, dashboard, mobile app, dan rencana hybrid AI/deep learning.

## Batasan Penting

Project ini:

- bukan alat diagnosis medis,
- bukan pengganti CTG/tocotransducer/Doppler,
- belum boleh mengklaim validasi klinis tanpa data uji,
- harus diposisikan sebagai sistem skrining awal dan pendukung pemantauan.

## Workflow AI

4. Implementasi satu issue
